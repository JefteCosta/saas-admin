# SaaS Admin

Sistema de exemplo (boilerplate) de aplicação SaaS multi-tenant construído com AdonisJS, Vue 3, Inertia.js e shadcn-vue.

## Stack

- **Backend**: AdonisJS v7 com TypeScript
- **Frontend**: Vue 3 + Inertia.js + Vite
- **UI**: shadcn-vue + Tailwind CSS v4
- **Banco**: SQLite (desenvolvimento) via Lucid ORM
- **Auth**: Session-based com @adonisjs/auth

## Começando

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev
```
