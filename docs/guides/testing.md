# Guia de Testes

Consulte este documento antes de adicionar, alterar ou executar testes.

## Diretrizes

- Adicione testes focados para mudanças comportamentais.
- Amplie cobertura ao tocar comportamento compartilhado, contratos entre módulos ou fluxos visíveis ao usuário.
- Se um comando não puder ser executado, documente o motivo nas notas finais da implementação.
