# Guia de Deploy

Consulte este documento antes de alterar deploy, release ou comportamento de ambiente de execução.

## Estado atual

O alvo de deploy (ambiente de hospedagem) ainda não está definido. O **fluxo de release** (versionamento, changelog e publicação via release-please) está documentado em [Guia de releases e versionamento](releases.md).

## Diretrizes

- Mantenha requisitos de ambiente alinhados com `.env.example`.
- Documente variáveis obrigatórias antes de depender delas em produção.
- Atualize este guia quando o ambiente de hospedagem for definido.
