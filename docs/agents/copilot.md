# Notas do GitHub Copilot

Consulte este documento ao usar o GitHub Copilot neste repositório.

## Diretrizes

- Use `.github/copilot-instructions.md` como entrada específica do Copilot.
- Use `AGENTS.md` e `docs/` como fontes compartilhadas de contexto.
- Agentes especializados ficam em `.github/agents/` e devem receber escopos de arquivos sem
  sobreposição.
- Siga `docs/agents/multi-agent.md` ao distribuir frentes independentes.
