# Glossário

Consulte este documento quando termos do projeto ou do domínio precisarem de definições consistentes.

## Termos

- **SaaS Admin**: este repositório e a aplicação associada; um sistema de exemplo de aplicação SaaS.
- **Tenant**: organização ou empresa que utiliza a plataforma; unidade de isolamento de dados.
- **Multi-tenancy**: capacidade de servir múltiplas organizações com isolamento de dados numa única instância.
- **Boilerplate**: código-base pré-configurado que serve como ponto de partida para novos projetos.
- **RBAC**: Role-Based Access Control; controle de acesso baseado em papéis (admin, membro, etc.).
- **Agente de IA**: assistente automatizado capaz de consultar contexto do projeto e apoiar tarefas de implementação, revisão, documentação ou planejamento.
- **Base de conhecimento**: documentação curada do projeto, usada por humanos e agentes de IA.
