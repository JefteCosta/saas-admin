# Visão Geral do Projeto

Consulte este documento quando precisar entender rapidamente o que este produto é e por que ele existe.

## O que é

**SaaS Admin** é um sistema de exemplo (boilerplate) de aplicação SaaS. Ele demonstra padrões e práticas comuns para construir aplicações SaaS multi-tenant modernas, incluindo:

- Autenticação e gerenciamento de sessões.
- Cadastro e gerenciamento de usuários.
- Estrutura preparada para multi-tenancy.
- Painel administrativo com sidebar, navegação e componentes reutilizáveis.
- Arquitetura full-stack com AdonisJS, Vue e Inertia.

## Objetivo

Servir como ponto de partida para aplicações SaaS, fornecendo a fundação técnica já implementada para que o desenvolvedor possa focar nas funcionalidades de negócio específicas do seu produto.

## Estado atual

O projeto possui a fundação básica implementada:
- Autenticação completa (signup, login, logout).
- Model de User com sessão.
- Layout com sidebar, navegação e estrutura de páginas.
- Componentes UI baseados em shadcn-vue.
