# Referências Técnicas

Consulte este documento quando precisar localizar documentação externa usada como referência técnica do projeto.

As referências desta pasta complementam a documentação autoral em `docs/`. Elas devem ser usadas para consultar APIs, convenções e comportamento de ferramentas externas, sem substituir as decisões específicas deste projeto.

## Referências disponíveis

- `docs/references/adonisjs/`: documentação de referência do AdonisJS.
- `docs/references/lucid/`: documentação de referência do Lucid.

## Como usar

- Consulte AdonisJS antes de alterar rotas, controllers, middleware, configuração, deploy ou estrutura de aplicação.
- Consulte Lucid antes de alterar migrations, models, queries, relacionamentos, factories ou seeders.
- Consulte `docs/architecture/overview.md` para entender como essas ferramentas são usadas neste projeto.

## Idioma

A documentação deste repositório deve permanecer em pt_BR. Quando uma referência externa for adicionada em outro idioma, ela deve ser traduzida ou acompanhada de uma versão em pt_BR antes de ser considerada parte da base de conhecimento principal.
