## Sobre o projeto

**SaaS Admin** é um sistema de exemplo (boilerplate) de aplicação SaaS multi-tenant. O objetivo é servir como base para construir aplicações SaaS completas, demonstrando padrões como autenticação, gerenciamento de usuários, isolamento de tenants e painel administrativo.
