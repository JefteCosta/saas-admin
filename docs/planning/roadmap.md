# Roadmap

Consulte este documento ao planejar próximos marcos técnicos ou de produto.

## Etapa atual

- Fundação SaaS básica com autenticação (signup, login, logout).
- Estrutura de frontend com layout sidebar e componentes shadcn-vue.
- Infraestrutura de documentação e workflows de CI/CD.

## Próximos marcos

1. **Multi-tenancy**: implementar model de Tenant, membership e isolamento de dados.
2. **RBAC**: papéis (owner, admin, member) com controle de permissões.
3. **Planos e limites**: configuração de limites por tenant/plano.
4. **Painel admin**: dashboard com métricas, CRUD de usuários, gerenciamento de papéis.
5. **Recursos SaaS**: convites, onboarding, configurações de conta, logs de auditoria.
6. **Documentação de extensão**: guias para adicionar módulos sobre a base.
